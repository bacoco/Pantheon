## Performance Validation

### Metrics to Check
```yaml

  response_time: < 200ms (p95)
  throughput: > 1000 req/s
  error_rate: < 0.1%
  cpu_usage: < 70%
  memory_usage: < 80%
```

### Performance Anti-patterns
- N+1 queries
- Synchronous blocking operations
- Memory leaks
- Unbounded growth
- Missing caching
